Filtration.Model — Type

Type of a Model model, composed by:

  • e₊ : the function $e_+(x_1)$
  • e₋ : the function $e_-(x_1)$
  • f₊ : the function $f_+(x_1)$
  • f₋ : the function $f_-(x_1)$
  • g₊ : the function $g_+(x_1)$
  • g₋ : the function $g_-(x_1)$
  • criteria_type : the criteria type, either $:min$ or $:max$
  • λ₀ : the constant associated to the criteria type
  • dynamics! : the cost and state dynamics function
  • de₊ : the function $e_+'(x_1)$, compute thanks to ForwardDiff package
  • de₋ : the function $e_-'(x_1)$, compute thanks to ForwardDiff package
  • df₊ : the function $f_+'(x_1)$, compute thanks to ForwardDiff package
  • df₋ : the function $f_-'(x_1)$, compute thanks to ForwardDiff package
  • dg₊ : the function $g_+'(x_1)$, compute thanks to ForwardDiff package
  • dg₋ : the function $g_-'(x_1)$, compute thanks to ForwardDiff package
  • xₛ : the singular state $m_s$
  • uₛ : the singular control $u_s$
  • Δx₂ : the value $\Delta_x = x_f - x_2(T)$, where $T$ is the final time to leave the singular arc

Filtration.Synthesis — Type

Type of a optimal synthesis, composed by:

  • model : a Model
  • tlim : the limits of time $t$
  • xlim : the limits of mass $m$
  • cut₋ : the cut₋ function
  • cut₊ : the cut₊ function
  • SL : the switching locus, as an ODESolution
  • DL : the dispersial locus, as Nothing if it not exists, and ODESolution if the method used to compute it is the continuation one, or a function if the method used is the direct one
