Filtration.Model — Type

Structure of a Model, composed by:

  • e₁ : the function $e_1(x_1)$
  • e₁ : the function $e_2(x_1)$
  • f₁ : the function $f_1(x_1)$
  • f₂ : the function $f_2(x_1)$
  • g₁ : the function $g_1(x_1)$
  • g₂ : the function $g_2(x_1)$
  • e₊ : the function $e_+(x_1)$
  • e₋ : the function $e_-(x_1)$
  • f₊ : the function $f_+(x_1)$
  • f₋ : the function $f_-(x_1)$
  • g₊ : the function $g_+(x_1)$
  • g₋ : the function $g_-(x_1)$
  • p : parameters of previous functions
  • criteria_type : the criteria type, either :min or :max
  • λ₀ : the constant associated to the criteria type
  • dynamics! : the cost and state dynamics function
  • de₊ : the function $e_+'(x_1)$, compute thanks to ForwardDiff package
  • de₋ : the function $e_-'(x_1)$, compute thanks to ForwardDiff package
  • df₊ : the function $f_+'(x_1)$, compute thanks to ForwardDiff package
  • df₋ : the function $f_-'(x_1)$, compute thanks to ForwardDiff package
  • dg₊ : the function $g_+'(x_1)$, compute thanks to ForwardDiff package
  • dg₋ : the function $g_-'(x_1)$, compute thanks to ForwardDiff package
  • xₛ : the singular state $x_s$
  • uₛ : the singular control $u_s$
  • λ₁ₛ : the singular costate $\lambda_{1,s}$ associated to the state $x_1$
  • λ₂ₛ : the singular costate $\lambda_{2,s}$ associated to the state $x_2$
  • Δx₂ : the value $\Delta_x = x_f - x_2(T)$, where $T$ is the final time to leave the singular arc

Filtration.Model — Method
Model(
    e₁::Function,
    e₂::Function,
    f₁::Function,
    f₂::Function,
    p::Vector{<:Real},
    criteria_type::Symbol;
    initial_guess
) -> Model

Constructor for Model structure.

Note

This function is a wrapper of Model function, where the functions $g_+$ and $g_-$ are set to 0 and 1 functions, respectively. This leads to $\dot x_2 = 1$ with $x_2(t_0) = 0$, ie. $x_2(t) = t$.

Arguments

  • e₊ : the function $e_+(x_1)$
  • e₋ : the function $e_-(x_1)$
  • f₊ : the function $f_+(x_1)$
  • f₋ : the function $f_-(x_1)$
  • p : parameters of previous functions
  • criteria_type : the criteria type, either :min or :max`

Keyword arguments (optional)

  • initial_guess : initial guess for the singular state $x_s$, initialized to 1.

Returns

  • model : the Model model

Filtration.solve_MP — Method
solve_MP(
    model::Model,
    init_x₂::Real,
    end_x₂::Real,
    init_x₁::Real;
    SL
) -> Solution

Solve the optimal control problem associated to a Model model by indirect method, with the structure given by $u_- \circ u_+$, and when the change is done by intersection with the switching curve

Warning

This function must be used only if we know the structure. If you don't know the structure of the solution, use solve function instead.

Arguments

  • model : a Model
  • init_x₂ : the initial state $x_1(t_0)$
  • end_x₂ : the final state $x_1(t_f)$
  • init_x₁ : the initial state $x_2(t_0)$

Returns

  • sol: the Solution
