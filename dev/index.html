Filtration.jl

Filtration.jl is a Julia package for solving optimal control problems arising in production–regeneration systems. It provides a high-level interface to define, analyze, and solve this specific class of control problems.

This package is designed to :

  • provide optimal control solutions
  • generate feedback synthesis
  • propose methods to construct a real-world control from optimal solutions
  • fit model with real-world data

Installation

Due to potential industrial partenerships, the code of the package is not accessible. If you are interested to try or use Filtration.jl package, please contact me.
